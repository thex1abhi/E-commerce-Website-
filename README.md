 RoyalWoods — Ecommerce Website

A modern Ecommerce Furniture Website built using Next.js, React, and Zustand for state management.
This project is a frontend-only application (no backend) that demonstrates cart management, product browsing, and responsive UI design.

🚀 Tech Stack 
   TypeScript
⚡ Next.js (App Router)
⚛️ React
🧠 Zustand — Global State Management
🎨 Tailwind CSS
📱 Responsive Design
🛒 Client-side Cart System
✨ Features
Modern ecommerce UI
Product browsing
Shopping cart functionality
Quantity management
Order summary calculation
Persistent global state (Zustand)
Fully responsive layout
Clean component architecture


The homepage introduces the brand with a hero section and product categories like Living Room, Dining Room, and Bedroom.

Highlights
Hero banner
Category cards
Minimal interior design layout
Smooth UI styling
🛍️ Shop Page

Displays available furniture products with clean card-based layouts.

Features
Product listing
Add to cart functionality
Responsive grid layout 

🛒 Cart Page

Users can manage items added to the cart and view order summaries.

Features

Remove items

Automatic subtotal & total calculation
🔐 Login Page

Simple UI login interface (frontend only).

Features
Clean authentication UI

Minimal footer design matching the white theme of the website.

Includes
Navigation links
